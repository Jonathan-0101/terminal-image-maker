# MacOS style terminal image generator

This is a simple tool that can be used to quickly create custom images in the style of a MacOS terminal.

## Config options
- Terminal size
- Terminal title
- Font and text size
- Prompt (pre-set options and custom)
- Colour
	- Prompt colour
	- Seperate command colour
	- Text colour
	- Terminal background colour
- Custom 'user' and 'host' which are re-used
